### Quick Start Guide – Versioning & Indexing Library

---

#### 1. What this system does (one paragraph)

This library is the **official home** for all policies, regulations, standards, and similar documents. It stores every version, tracks changes, and lets apps reliably fetch the **right text**—the latest version or a specific historical one—down to individual sections like “§3.2.1(a)”. If your app needs to show “the official policy” or link to “this exact clause,” this is the system you talk to.

---

#### 2. Core ideas in 60 seconds

- **Document:** A policy, regulation, law, etc.  
- **Version:** A specific state of that document (e.g. `1.0.0`, `1.1.0`).  
- **CDI:** Canonical Document Identifier (permanent ID for a document family).  
- **Sections:** Structured parts of a document (e.g. `§3.2.1(a)`).  
- **Amendment:** A change to one or more sections.  

---

#### 3. Typical things you’ll do as a developer

- **Show the latest official version** of a document in your app.  
- **Link to a specific section** (e.g. a clause in a policy).  
- **Check what the policy said on a past date** (for audits).  
- **Search** for documents by keywords, type, or jurisdiction.  

---

#### 4. The three most important endpoints

1. **Get the latest version of a document**

   ```http
   GET /v1/documents/{cdi}
   ```

   - Input: `cdi` (e.g. `US-WYDOT-POL-0042`)  
   - Output: full document version (metadata, sections, etc.)

2. **Get all versions of a document**

   ```http
   GET /v1/documents/{cdi}/versions
   ```

   - Use this to see history and pick a specific version.

3. **Resolve a specific section (citation)**

   ```http
   GET /v1/citations/resolve?cdi={cdi}&index_code={section_code}&as_of_date={date?}
   ```

   - Example: `index_code=§3.2.1(a)`  
   - Optional `as_of_date` to see what was in force on a given day.

---

#### 5. Minimal integration example (JavaScript)

```javascript
async function showPolicy(cdi) {
  const res = await fetch(`/v1/documents/${cdi}`);
  if (!res.ok) throw new Error("Document not found");
  const doc = await res.json();
  document.querySelector("#title").textContent = doc.title;
  document.querySelector("#body").textContent = doc.sections[0].text;
}
```

---

#### 6. Minimal integration example (Python)

```python
import requests

def get_latest_document(cdi):
    r = requests.get(f"https://api.example.com/v1/documents/{cdi}")
    r.raise_for_status()
    return r.json()

doc = get_latest_document("US-WYDOT-POL-0042")
print(doc["title"])
```

---

#### 7. Golden rules

- **Always store the CDI**, not just the title.  
- **Don’t copy the text into your own DB** if you can avoid it—fetch it.  
- **Use section index codes** (`§3.2.1(a)`) for deep links.  
- **Use `as_of_date`** when you care about what was true at a specific time.

### Glossary for Non-Technical & Technical Users

---

#### A. Core terms

- **Document**  
  An official text such as a policy, regulation, law, standard, or directive.

- **Document Version**  
  A specific state of a document at a point in time (e.g. `1.0.0`).

- **Version Number (Semantic Versioning)**  
  A three-part number:  
  - **MAJOR**: big changes (new framework, repeal-and-replace).  
  - **MINOR**: meaningful changes (new sections, updated rules).  
  - **PATCH**: small fixes (typos, formatting).

- **CDI (Canonical Document Identifier)**  
  A permanent, unique ID for a document family.  
  Example: `US-WYDOT-POL-0042-v3.1`.

- **Series ID**  
  A stable ID for the document family (e.g. `0042`), used across versions.

---

#### B. Structure & navigation

- **Section**  
  A part of a document, like a chapter or clause.

- **Section Index Code**  
  A standardized way to refer to a section.  
  Example: `§3.2.1(a)(iv)`.

- **Hierarchy**  
  The levels inside a document, such as:  
  `Part → Chapter → Section → Subsection → Clause → Item`.

---

#### C. Change tracking

- **Amendment**  
  A change to one or more sections of a document (add, modify, repeal, etc.).

- **Amendment Operation**  
  The type of change: `ADD`, `MODIFY`, `REPEAL`, `RENUMBER`, `REPLACE`.

- **Supersedes**  
  Indicates that one version replaces another.

- **Effective Date**  
  The date when a document or amendment starts to have effect.

- **Expiration Date**  
  The date when a document stops being in force.

---

#### D. Status & lifecycle

- **DRAFT**  
  Not yet approved or official.

- **ACTIVE**  
  Currently in force and authoritative.

- **RETIRED**  
  No longer in use but kept for history.

- **REPEALED**  
  Officially revoked.

- **SUPERSEDED**  
  Replaced by a newer version.

---

#### E. Integrity & authenticity

- **Checksum**  
  A digital fingerprint of a document version used to detect tampering.

- **Digital Signature**  
  A cryptographic proof that a document version was approved by the right authority and hasn’t been altered.

- **Audit Log**  
  A record of who did what, when (e.g. created, approved, published).

---

#### F. Technical terms

- **API (Application Programming Interface)**  
  A set of URLs and rules that software uses to talk to the library.

- **REST**  
  A common style of API using HTTP verbs like GET, POST, etc.

- **JSON**  
  A text format for structured data, easy for both humans and machines.

- **URI**  
  A unique address for a resource (e.g. where a document file is stored).

### SDK – Example Client Libraries

I’ll sketch **two lightweight SDKs**: one in **Python** and one in **JavaScript**. Both assume a base URL like `https://api.example.com`.

---

#### Python SDK (simple, extendable)

```python
# sdk/document_client.py
import requests
from typing import Optional, Dict, Any, List

class DocumentClient:
    def __init__(self, base_url: str, api_key: Optional[str] = None):
        self.base_url = base_url.rstrip("/")
        self.api_key = api_key

    def _headers(self) -> Dict[str, str]:
        headers = {"Accept": "application/json"}
        if self.api_key:
            headers["Authorization"] = f"Bearer {self.api_key}"
        return headers

    def get_latest_document(self, cdi: str) -> Dict[str, Any]:
        url = f"{self.base_url}/v1/documents/{cdi}"
        r = requests.get(url, headers=self._headers())
        r.raise_for_status()
        return r.json()

    def get_document_versions(self, cdi: str) -> List[Dict[str, Any]]:
        url = f"{self.base_url}/v1/documents/{cdi}/versions"
        r = requests.get(url, headers=self._headers())
        r.raise_for_status()
        return r.json()

    def get_specific_version(self, cdi: str, version: str) -> Dict[str, Any]:
        url = f"{self.base_url}/v1/documents/{cdi}/versions/{version}"
        r = requests.get(url, headers=self._headers())
        r.raise_for_status()
        return r.json()

    def resolve_citation(
        self,
        cdi: str,
        index_code: str,
        as_of_date: Optional[str] = None,
    ) -> Dict[str, Any]:
        params = {"cdi": cdi, "index_code": index_code}
        if as_of_date:
            params["as_of_date"] = as_of_date
        url = f"{self.base_url}/v1/citations/resolve"
        r = requests.get(url, headers=self._headers(), params=params)
        r.raise_for_status()
        return r.json()

    def search_documents(
        self,
        query: str,
        type: Optional[str] = None,
        jurisdiction: Optional[str] = None,
        status: Optional[str] = None,
    ) -> Dict[str, Any]:
        params = {"q": query}
        if type:
            params["type"] = type
        if jurisdiction:
            params["jurisdiction"] = jurisdiction
        if status:
            params["status"] = status
        url = f"{self.base_url}/v1/search"
        r = requests.get(url, headers=self._headers(), params=params)
        r.raise_for_status()
        return r.json()
```

**Usage:**

```python

client = DocumentClient("https://api.example.com", api_key="YOUR_TOKEN")

doc = client.get_latest_document("US-WYDOT-POL-0042")
```

#### JavaScript SDK (browser or Node)

```javascript
// sdk/documentClient.js
export class DocumentClient {
  constructor(baseUrl, apiKey = null) {
    this.baseUrl = baseUrl.replace(/\/$/, "");
    this.apiKey = apiKey;
  }

  _headers() {
    const headers = { Accept: "application/json" };
    if (this.apiKey) {
      headers["Authorization"] = `Bearer ${this.apiKey}`;
    }
    return headers;
  }

  async _get(path, params = {}) {
    const url = new URL(this.baseUrl + path);
    Object.entries(params).forEach(([k, v]) => {
      if (v !== undefined && v !== null) url.searchParams.append(k, v);
    });

    const res = await fetch(url.toString(), {
      method: "GET",
      headers: this._headers(),
    });

    if (!res.ok) {
      const text = await res.text();
      throw new Error(`Request failed: ${res.status} ${text}`);
    }
    return res.json();
  }

  getLatestDocument(cdi) {
    return this._get(`/v1/documents/${encodeURIComponent(cdi)}`);
  }

  getDocumentVersions(cdi) {
    return this._get(`/v1/documents/${encodeURIComponent(cdi)}/versions`);
  }

  getSpecificVersion(cdi, version) {
    return this._get(
      `/v1/documents/${encodeURIComponent(cdi)}/versions/${encodeURIComponent(
        version
      )}`
    );
  }

  resolveCitation(cdi, indexCode, asOfDate = null) {
    const params = { cdi, index_code: indexCode };
    if (asOfDate) params.as_of_date = asOfDate;
    return this._get("/v1/citations/resolve", params);
  }

  searchDocuments({ q, type, jurisdiction, status }) {
    return this._get("/v1/search", { q, type, jurisdiction, status });
  }
}
```

**Usage:**

```javascript
import { DocumentClient } from "./sdk/documentClient.js";

const client = new DocumentClient("https://api.example.com", "YOUR_TOKEN");

async function demo() {
  const doc = await client.getLatestDocument("US-WYDOT-POL-0042");
  console.log(doc.title);

  const section = await client.resolveCitation(
    "US-WYDOT-POL-0042",
    "§3.2.1(a)"
  );
  console.log(section.text);
}

demo();
```
